This project details how to train kids computers. The suggestion is that before you teach kids how to use applications in the computer, first teach them to type. Why? If we teach kids to type, in this case touch typing they become comfortable with the computer. Touch typing can be thought of as training the fingers to adjust to the keyboard, so that kids can work faster on the computer. The faster they can work on the computer the more enjoyable the experience. The more enjoyable the experience the more they would want to work on the computer.

Of course touch typing cannot exist in isolation. It should be paired with learning English. All four ways: reading, writing, listening and speaking. More emphasis should be placed on writing of course. The idea is to get kids to be typing as much as possible so that they can fine tune their writing ability. Lessons to start with include sentence construction.

This is a far more inclusive approach as it doesn't matter whether you are using a powerful computer or not, typing is fundamental to learning computers more effeectively. In an extreme case, you can teach a child typing then teach them VIM to use for writing

- Type at speed of thought
- Practice based learning
- Writing as thinking
- Writing through typing
- Stimulate thinking (Sentence Construction, Recalling events, Narration)
